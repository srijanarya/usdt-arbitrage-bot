# Treum Algotech - Binance P2P Merchant Setup Guide

## Company Registration & Gold Status Path

### Step 1: Create Business Binance Account
1. Go to: https://www.binance.com/en/support/faq/how-to-register-a-corporate-account-on-binance-63045df5bf9c4e57a6e1f1a9cf9b7a51
2. Select "Entity Account" (not personal)
3. Use company email: admin@treumalgotech.com (or create one)

### Step 2: Business Documentation Required
- **Company Registration Certificate**
- **Articles of Association**
- **Director's ID Proof (Aadhaar/PAN)**
- **Board Resolution for Crypto Trading**
- **Proof of Business Address**
- **Bank Statement (Company Account)**

### Step 3: Complete Enhanced KYC
1. Basic Information
2. Advanced Verification
3. Enterprise Verification
4. Address Proof Verification

### Step 4: P2P Merchant Application

#### Eligibility Requirements:
- [ ] 30 days since registration
- [ ] Identity verification completed
- [ ] 2FA enabled
- [ ] Email verified
- [ ] Complete 10+ P2P trades

#### Fast Track Option:
- **Security Deposit**: 5000 USDT
- **Completion Rate**: >95%
- **Response Time**: <15 min average

### Step 5: Gold Merchant Status Requirements

#### Trading Volume (Monthly):
- Bronze: 10,000 - 50,000 USDT
- Silver: 50,000 - 200,000 USDT
- Gold: 200,000+ USDT

#### Performance Metrics:
- Completion Rate: >98%
- Release Time: <10 minutes
- Disputes: <0.5%
- Active Days: 20+ per month

### Quick Start Strategy:

1. **Week 1-2**: 
   - Complete business verification
   - Deposit 5000 USDT security
   - Complete 10 small trades as buyer

2. **Week 3-4**:
   - Apply for merchant status
   - Start with small ads (100-500 USDT)
   - Focus on completion rate

3. **Month 2**:
   - Scale to 1000+ USDT ads
   - Maintain <10 min release time
   - Target 10,000 USDT volume

4. **Month 3**:
   - Push for 50,000+ USDT volume
   - Apply for Silver status
   - Optimize pricing algorithm

5. **Month 4-6**:
   - Scale to 200,000+ USDT monthly
   - Achieve Gold status
   - Unlock premium features

### Benefits of Gold Status:
- Priority customer support
- Higher posting limits
- Better visibility in listings
- Access to bulk trading tools
- Reduced trading fees
- Exclusive merchant events
