Build Your USDT Arbitrage Bot with AI-Powered Development

üéØ Goal: Get your arbitrage bot running in Cursor within 30 minutes using AI commands!

1Open Terminal in Cursor

Press Ctrl + ` to open the integrated terminal

cd /Users/srijan/Desktop/usdt-arbitrage-bot ./quick-setup.sh

This installs Node.js and PostgreSQL if needed (takes 2-3 minutes)

2Initialize Project

Still in terminal, run these commands:

npm init -y npm install express axios ws pg dotenv npm install -D typescript @types/node @types/express @types/ws @types/pg nodemon ts-node

3Create Project Structure

mkdir -p src/{api/exchanges,services,config,models,utils} touch src/index.ts .env .gitignore tsconfig.json

4Generate tsconfig.json with AI

Open tsconfig.json and press Cmd + K

Create a TypeScript configuration for Node.js with ES2020 target, strict mode, and proper module resolution for a crypto trading bot

5Create Main Server File

Open src/index.ts and press Cmd + K

Create an Express + WebSocket server for crypto arbitrage with health endpoints, error handling, and graceful shutdown

6Generate Exchange Clients

Create src/api/exchanges/coinDCX.ts and use Cmd + K:

Create CoinDCX API client with WebSocket for USDT/INR real-time prices, HMAC auth, reconnection logic, and rate limiting

Then create src/api/exchanges/zebPay.ts:

Create ZebPay API client with REST polling for USDT/INR, authentication, and rate limiting

7Build Arbitrage Scanner

Create src/services/priceScanner.ts and press Cmd + K:

Create arbitrage scanner that monitors multiple exchanges, detects opportunities after fees and 1% TDS, stores in PostgreSQL, and emits events

8Setup Environment

Create .env file:

# Database DB_HOST=localhost DB_PORT=5432 DB_NAME=arbitrage_bot DB_USER=postgres DB_PASSWORD= # APIs (get from exchange websites) COINDCX_API_KEY= COINDCX_API_SECRET= ZEBPAY_API_KEY= ZEBPAY_API_SECRET= # Trading MIN_PROFIT_THRESHOLD=100 PORT=3000

9Add NPM Scripts

In package.json, update the scripts section:

"scripts": { "dev": "nodemon --exec ts-node src/index.ts", "build": "tsc", "start": "node dist/index.js" }

10Start Development!

npm run dev

Your server should start on port 3000! üéâ
